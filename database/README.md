# Database

Esta carpeta contiene scripts SQL de inicializacion y mantenimiento.

- `modelo-datos/UP.sql`: crea el modelo.
- `modelo-datos/SEED.sql`: carga datos base.
- `modelo-datos/DOWN.sql`: elimina el modelo.
- `modelo-datos/seed/`: modulos de seed separados por responsabilidad.
- `modelo-datos/ops/`: scripts operativos/manuales (purga, parches, cargas puntuales).
